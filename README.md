# Verbal Assault

Simple Python CLI application that generates random phrases.

## Project structure

src/ – main application code
data/ – phrase dataset
assets/ – icons and resources

## Installation

```bash
git clone https://github.com/GhostDainy/verbal-assault.git
cd verbal-assault
```

## Run

```bash
python src/verbal_assault.py
```

## Description

This CLI program randomly selects phrases from a dataset and prints them to the console.
